### pass-gen-v2

Пет-приложение с примером консольного интерактивного приложения на spring boot.

#### Сборка

Сборка с graalvm нативного образа для получения исполняемого файла
```shell
mvn clean native:compile -Pnative -DskipTests
```

#### Использование

- Получение справки
```shell
help 
```

В данный момент реализована одна команда `create`. Получить справку по ней можно при помощи следующей команды
```shell
create --help
```
